Memulai InventoryPro
Daftar, login, dan setup awal
1
Buka Aplikasi
Buka file index.html di browser (Chrome / Safari disarankan). Pastikan koneksi internet aktif karena data disimpan di Firebase Cloud.
2
Daftar Akun Baru
Ketuk tab ✨ Daftar → isi Nama Depan, Nama Bisnis, Email, Password (min. 8 karakter) → centang Syarat & Ketentuan → Ketuk Buat Akun.
3
Login
Masukkan email dan password → Ketuk 🔐 Masuk. Gunakan Lupa Password jika lupa — link reset dikirim ke email.
4
Setup Info Toko
Ketuk menu 👤 Akun → isi Nama Toko, Alamat, Telp, Logo → Simpan. Info ini akan muncul di semua struk cetak.
5
Pasang PIN Admin
Di halaman Akun → Keamanan Admin → set PIN 4–6 digit untuk melindungi aksi edit/hapus produk.
💡 Setiap akun memiliki database yang sepenuhnya terisolasi — pengguna lain tidak bisa melihat data Anda.

Manajemen Produk & Bin
Tambah, edit, hapus produk, kelola lokasi penyimpanan

Produk diorganisir dalam Bin — lokasi fisik penyimpanan (contoh: A1, RAK-B2, COLD-1).

1
Tambah Produk Baru
Ketuk tombol ➕ Tambah di halaman Produk → isi SKU (kode unik), Nama, Harga, Minimum Stok → Simpan.
2
Scan Barcode SKU
Ketuk 📷 di kolom cari → arahkan kamera ke barcode produk → SKU otomatis terisi dan produk ditemukan.
3
Tab Bin View
Tampilkan semua lokasi penyimpanan. Ketuk nama bin untuk melihat isi dan melakukan Transfer Bin.
4
Edit / Hapus Produk
Ketuk produk → ketuk ✏️ Edit atau 🗑️ Hapus → wajib masukkan PIN Admin untuk melanjutkan.
⚠️ SKU harus unik — tidak boleh ada dua produk dengan SKU sama. Gunakan format konsisten misal PRD-001.

🖨️ Cetak Struk
1
Scan / Cari Produk
Ketuk 📷 untuk scan barcode atau ketik SKU secara manual. Harga jual otomatis terisi dari data produk (bisa diubah).
2
Tambah ke Keranjang
Atur Qty → ketuk ➕ Tambah. Bisa tambah banyak produk berbeda dalam satu transaksi. Gunakan tombol +/- untuk ubah qty di keranjang.
3
Input Pembayaran
Ketuk Pas (bayar pas), 50rb, atau 100rb sebagai shortcut. Kembalian dihitung otomatis.
4
Proses Bayar & Cetak
Ketuk 💳 Bayar → struk muncul langsung → ketuk 🖨️ Cetak Struk untuk print ke printer thermal.

Reprint Sales
Cetak ulang struk transaksi yang sudah terjadi
1
Dari Tombol Header POS
Di halaman POS, ketuk tombol 🧾 Reprint di pojok kanan atas (sejajar nomor invoice) → muncul daftar semua invoice.
2
Dari Riwayat Hari Ini
Scroll ke bawah halaman POS → bagian 📋 Riwayat POS → setiap invoice ada tombol 🖨️ Reprint di sebelah kanan.

Ketuk tombol 🖨️ Print pada invoice yang diinginkan → preview struk muncul → ketuk 🖨️ Print Thermal → jendela print terbuka.

Reset Data
Hapus data secara massal — HANYA untuk Owner
🚨 PERINGATAN: Reset data bersifat permanen dan tidak dapat dibatalkan. Pastikan sudah export data sebelum melakukan reset.
Tombol ResetYang Dihapus
🗑️ Reset Semua TransaksiSeluruh riwayat transaksi (IN/OUT/POS/MOVE). Stok tidak berubah.
📦 Reset Semua StokSeluruh data inventori/stok di semua bin. Produk tetap ada.
🗂️ Reset Semua ProdukSeluruh produk dan stok. Transaksi tetap ada.
💣 RESET TOTALSEMUA data: produk, stok, dan transaksi sekaligus.

Prosedur Reset

1
Export data ke Excel terlebih dahulu (Laporan → 📊 Export Excel)
2
Buka Akun → Zona Berbahaya → pilih jenis reset
3
Muncul dialog konfirmasi → ketik HAPUS (huruf kapital) → klik OK
4
Masukkan PIN Admin → proses penghapusan berjalan

🔧 Setup Printer Thermal

1
Pastikan printer thermal terhubung dan driver terinstal. Printer yang umum: Xprinter, Epson TM-T82, RP-58.
2
Saat dialog print muncul di browser, pilih printer thermal Anda. Ukuran kertas otomatis sudah di-set ke 58mm × auto.
3
Set margin ke 0 di dialog print browser untuk hasil terbaik.
🔔 Jika tombol print tidak membuka jendela baru, browser mungkin memblokir popup. Cari ikon di address bar dan izinkan popup untuk situs ini.

FAQ — Pertanyaan Umum
Solusi masalah yang sering ditemui
Aplikasi stuck / tidak bisa dioperasikan saat pertama dibuka ▼
Pastikan koneksi internet aktif. Aplikasi membutuhkan internet untuk terhubung ke Firebase. Coba refresh halaman (F5). Jika masih bermasalah, coba buka di browser lain (Chrome direkomendasikan).
Tombol print tidak membuka jendela cetak ▼
Browser Anda memblokir popup. Lihat ikon di address bar (biasanya ikon popup diblokir) → klik → pilih "Izinkan popup dari situs ini". Kemudian coba print lagi.
Lupa PIN Admin, bagaimana cara reset? ▼
Saat ini tidak ada reset PIN otomatis. Hubungi developer untuk reset manual via Firebase Console dengan menghapus field adminPin di dokumen user Anda di Firestore.
Stok tidak berkurang setelah transaksi POS ▼
Pastikan produk memiliki stok di bin tertentu. Jika qty di bin = 0, stok tidak bisa dikurangi. Lakukan Stok Masuk terlebih dahulu.
Apakah data saya aman? Bisakah pengguna lain melihat data saya? ▼
Tidak. Setiap akun memiliki database yang sepenuhnya terisolasi di Firestore (path users/{uid}/...). Firestore Security Rules memblokir semua akses lintas user secara server-side.
Bagaimana cara backup data? ▼
Buka menu 📋 Laporan → klik 📊 Export Excel. File .xlsx berisi semua transaksi dan ringkasan. Lakukan ini secara rutin (mingguan/bulanan) sebagai backup.
Kamera scan barcode tidak bisa digunakan ▼
Browser memerlukan izin kamera. Saat pertama kali klik 📷, browser akan meminta izin — klik Izinkan. Jika sudah diblokir sebelumnya, buka pengaturan browser → Site Settings → Camera → ubah ke Izinkan untuk situs ini.
Logo di struk tidak muncul saat print ▼
Logo disimpan sebagai base64 langsung di struk. Pastikan ukuran file gambar tidak melebihi 500KB. Jika melebihi batas, upload gagal dan logo tidak tersimpan. Kompres gambar terlebih dahulu di situs seperti tinypng.com.
